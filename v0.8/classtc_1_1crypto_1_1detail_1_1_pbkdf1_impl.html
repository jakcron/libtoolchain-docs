Detailed Description

template<typename HashFunction>
class tc::crypto::detail::Pbkdf1Impl< HashFunction >

This class implements Password-Based Key Derivation Function 1 (PBKDF1)

Template Parameters
HashFunctionThe class that implements the hash function used for key derivation.

PBKDF1 is a hash based key derivation function, as defined in RFC 8018. Applicable hash functions to use with PBKDF1 include.

  1. MD4
  2. MD5
  3. SHA1
