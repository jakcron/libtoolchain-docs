Detailed Description

template<size_t KeySize>
class tc::crypto::AesEncryptor< KeySize >

Class for AES encryption/decryption.

Template Parameters
KeySizeSize in bytes of the AES encryption key. This must be 16, 24 or 32.

This class has three states:

  • None : Not ready
  • Initialized : Ready to process data

General usage of this class is as follows:

  • Initialize AES state with initialize().
  • Encrypt or decrypt block(s) using encrypt() or decrypt().
