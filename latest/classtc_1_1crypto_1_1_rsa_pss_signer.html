Detailed Description

template<size_t KeyBitSize, class HashFunction>
class tc::crypto::RsaPssSigner< KeyBitSize, HashFunction >

Class for calculating an RSA-PSS signature.

Template Parameters
KeyBitSizeRSA key size in bits.
HashFunctionThe class that implements the hash function used with RSA-PSS.

This class is a template class that takes a hash function implementation class as template parameter. See Rsa2048PssSha2256Signer or similar for supplied realizations of this template class.

The KeyBitSize is the size in bits of the RSA key, this only supports key sizes aligned to 8 bits.

The implementation of HashFunction must satisfies the following conditions. See Sha2256Generator or similar class, for more information including parameters to each function.

  1. Has a kBlockSize constant that defines the size of the block to process.
  2. Has a kHashSize constant that defines the output size of the hash value.
  3. Has an initialize method that begins processing.
  4. Has an update method that updates the hash value on input.
  5. Has a getHash method that gets the final hash value.

This class has two states:

  • None : Not ready
  • Initialized : Ready to process input data

General usage of this class is as follows:

  • Initialize RSA Signer state with initialize().
  • Sign/Verify message digest with sign() / verify().

[in]saltPointer to salt.
[in]salt_sizeSize of salt.
Returns
true if signature calculation was successful.
Precondition
  • Size of the signature buffer must >= Rsa1024PssSha2256Signer::kSignatureSize.
  • The data in salt should be random, otherwise the signature strength is reduced.
  • Where KeySize >= (HashCalculator::kHashSize * 2 + 2) salt_size = HashCalculator::kHashSize. Otherwise salt_size = KeySize - (HashCalculator::kHashSize + 2). However the minimum legal salt size is (HashCalculator::kHashSize - 2), if the salt_size falls below this consider a larger KeySize as this operation will not complete successfully.
Postcondition
  • The signature is written to signature.

This function calculates a signature for a message digest.
