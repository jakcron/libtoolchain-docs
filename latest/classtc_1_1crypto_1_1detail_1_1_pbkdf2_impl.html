Detailed Description

template<typename HashFunction>
class tc::crypto::detail::Pbkdf2Impl< HashFunction >

This class implements Password-Based Key Derivation Function 2 (PBKDF2)

Template Parameters
HashFunctionThe class that implements the hash function used for key derivation.

PBKDF2 is a hmac based key derivation function, as defined in RFC 8018. Applicable hash functions to use with PBKDF2 include.

  1. SHA1
  2. SHA2-224
  3. SHA2-256
  4. SHA2-384
  5. SHA2-512
