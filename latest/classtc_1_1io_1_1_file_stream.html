Open file stream.

Parameters
[in]pathA relative or absolute path for the file that the current FileStream object will encapsulate.
[in]modeOne of the enumeration values that determines how to open or create the file.
[in]accessOne of the enumeration values that determines how the file can be accessed by the FileStream object. This also determines the values returned by the canRead and canWrite methods of the FileStream object. canSeek is true if path specifies a disk file.
Exceptions
tc::ArgumentExceptionpath contains invalid characters or is empty.
tc::NotSupportedExceptionpath refers to an unsupported non-file device.
tc::io::IOExceptionAn I/O error, such as specifying mode CreateNew when the file specified by path already exists, occurred. Or the stream has been closed.
tc::SecurityExceptionThe caller does not have the required permission.
tc::io::DirectoryNotFoundExceptionThe specified path is invalid, such as being on an unmapped drive.
tc::UnauthorisedAccessExceptionThe access requested is not permitted by the operating system for the specified path, such as when access is Write or ReadWrite and the file or directory is set for read-only access.
tc::io::PathTooLongExceptionThe specified path, file name, or both exceed the system-defined maximum length.
tc::ArgumentOutOfRangeExceptionmode contains an invalid value.
tc::ArgumentOutOfRangeExceptionaccess contains an invalid value.

size_t tc::io::FileStream::read ( byte_t *  ptr,
size_t  count 
)
virtual

Reads a sequence of bytes from the current stream and advances the position within the stream by the number of bytes read.

Parameters
[out]ptrPointer to an array of bytes. When this method returns, ptr contains the specified byte array with the values between 0 and (count - 1) replaced by the bytes read from the current source.
[in]countThe maximum number of bytes to be read from the current stream.
Returns
The total number of bytes read into ptr. This can be less than the number of bytes requested if that many bytes are not currently available, or zero (0) if the end of the stream has been reached.
Precondition
A stream must support reading for read to work.
Note
Use canRead to determine if this stream supports reading.
Exceptions
tc::AccessViolationptr refers to inaccessible/protected memory.
tc::ArgumentNullExceptionptr is nullptr.
tc::ArgumentOutOfRangeExceptioncount is negative.
tc::io::IOExceptionAn I/O error occurred.
tc::NotSupportedExceptionThe stream does not support reading.
tc::ObjectDisposedExceptionMethods were called after the stream was closed.
