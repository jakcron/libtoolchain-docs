Detailed Description

This class implements a Psuedo Random Byte Generator.

The underlying algorithm is CTR_DBRG. This class generates random data suitable for encryption use cases.

  • Initialization vectors
  • Salts / Nonces
  • HMAC keys
  • AES keys
