Detailed Description

template<size_t KeyBitSize, class HashFunction>
class tc::crypto::RsaPkcs1Signer< KeyBitSize, HashFunction >

Class for calculating an RSA-PKCS1 signature.

Template Parameters
KeyBitSizeRSA key size in bits.
HashFunctionThe class that implements the hash function used with RSA-PKCS1.

This class is a template class that takes a hash function implementation class as template parameter. See Rsa2048Pkcs1Sha256Signer or similar for supplied realizations of this template class.

The KeyBitSize is the size in bits of the RSA key, this only supports key sizes aligned to 8 bits.

The implementation of HashFunction must satisfies the following conditions. See Sha256Generator or similar class, for more information including parameters to each function.

  1. Has a kAsn1OidDataSize constant that defines the size of the ASN.1 encoded OID for the hash algorithm
  2. Has a kAsn1OidData constant that defines the ASN.1 encoded OID for the hash algorithm
  3. Has a kBlockSize constant that defines the size of the block to process.
  4. Has a kHashSize constant that defines the output size of the hash value.
  5. Has an initialize method that begins processing.
  6. Has an update method that updates the hash value on input.
  7. Has a getHash method that gets the final hash value.

This class has two states:

  • None : Not ready
  • Initialized : Ready to process input data

General usage of this class is as follows:

  • Initialize RSA Signer state with initialize().
  • Sign/Verify message digest with sign() / verify().
