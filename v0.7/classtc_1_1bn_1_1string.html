Detailed Description

template<size_t ENCODED_SIZE, size_t LOGICAL_SIZE = ENCODED_SIZE>
class tc::bn::string< ENCODED_SIZE, LOGICAL_SIZE >

This class represents a literal char array.

Template Parameters
ENCODED_SIZELiteral size of the string structure. sizeof() will return this size.
LOGICAL_SIZELogical maximum size of the string, LOGICAL_SIZE <= ENCODED_SIZE.

The intended for use is for defining structures read from files.

Consider this structure read from a file with some comments from the spec:

struct MyStruct
{
uint32_t version;
char product_sku[16]; // ASCII but not NULL terminated, so 16 usable characters
char product_title[32]; // ASCII but must be NULL terminated, so only 31 usable chars
uint32_t product_version;
};

The member product_sku according to specification can have all 16 chars populated, and if all 16 chars are populated it isn't guarenteed to be null terminated. The member product_title according to specification must be null terminated, reserving one byte for the null byte. This isn't the same behaviour as product_sku, requiring different string logic specific to this one struct. But these constraints can be enforced into the defintion of MyStruct using tc::bn::string.

Consider this revised struct using tc::bn::string :

struct MyStruct
{
uint32_t version;
tc::bn::string<16> product_sku; // this has a ENCODED_SIZE & LOGICAL_SIZE of 16 bytes which mean's the size on disk is 16 bytes and there are 16 usable characters
tc::bn::string<32,31> product_title; // this has a ENCODED_SIZE of 32 bytes & LOGICAL_SIZE of 31 bytes which mean's the size on disk is 32 bytes and there are 31 usable characters
uint32_t product_version;
};

In the above struct the correct size of product_sku and product_title are preserved while also enforcing the logical size of the string.

To get the maximum length a string can be for a given tc::bn::string, use max_size():

MyStruct st;
size_t product_sku_max_size = st.product_sku.max_size(); // 16
size_t product_title_max_size = st.product_title.max_size(); // 31

To get the current string length for a given tc::bn::string, use size():

MyStruct st;
size_t product_sku_str_size = st.product_sku.size();
size_t product_title_str_size = st.product_title.size();

To decode the data in a tc::bn::string to a std::string, use decode():

MyStruct st;
std::string product_sku = st.product_sku.decode();
std::string product_title = st.product_title.decode();

To encode a std::string into a tc::bn::string, use encode():

MyStruct st;
st.product_sku.encode("SKU-1234-X");
st.product_title.encode("MyProductTitle");
