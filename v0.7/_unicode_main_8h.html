Detailed Description

Declaration of unicode entry point (umain())

Author
Jack (jakcron)
Version
0.1
Date
2020/04/13

Including UnicodeMain.h bootstraps umain() by defining the platform specific unicode entry point.

You must define umain() as your entry point if you include this header. Compiler errors will occur if you define any other entrypoints or do not define umain().
