Detailed Description

An interface for implementing a basic FileSystem handler.

Defines expected functionality including:

  • File access (open)
  • File management (create, remove)
  • Directory traversal (get current directory, change current directory)
  • Directory management (create, remove)
  • Directory listing
Note
IFileSystem uses the tc::io::Path class to represent a path, not as a literal string.
It is up to the implementation of IFileSystem to validate and process tc::io::Path objects.
It is up to the implementation to enforce tc::io::FileMode & tc::io::FileAccess.
