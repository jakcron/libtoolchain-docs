Detailed Description

A struct which contains vectors of directory and file entries, including mapping between absolute tc::io::Path and a dir/file entry.

FileSystemSnapshot is snapshot of a FileSystem directory tree. It only has to support read-only access to directory information and file streams.

VirtualFileSystem considers a directory to exist if the following conditions are satisfied:

  • The absolute tc::io::Path to the directory exists in dir_entry_path_map
  • The index from looking up the absolute path in dir_entry_path_map is a valid index for dir_entries

VirtualFileSystem considers a file to exist if the following conditions are satisfied:

  • The absolute tc::io::Path to the file exists in file_entry_path_map
  • The index from looking up the absolute path in file_entry_path_map is a valid index for file_entries
  • The FileEntry in file_entries has a stream member that is not nullptr
  • The IStream has properties: .canRead() == true, .canWrite() == false
