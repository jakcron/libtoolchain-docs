Detailed Description

template<class HashFunction>
class tc::crypto::Pbkdf1KeyDeriver< HashFunction >

Class for deriving a key using Password-Based Key Derivation Function 1 (PBKDF1).

Template Parameters
HashFunctionThe class that implements the hash function used for key derivation.

PBKDF1 is a hash based key derivation function, as defined in RFC 8018. As such this template class requires HashFunction to implement one of the following hash functions to be compliant with RFC 8018.

  1. MD4
  2. MD5 (see Md5Generator)
  3. SHA1 (see Sha1Generator)

The implementation of HashFunction must satisfies the following conditions. See Sha1Generator or similar class, for more information including parameters to each function.

  1. Has a kBlockSize constant that defines the size of the block to process.
  2. Has a kHashSize constant that defines the output size of the hash value.
  3. Has an initialize method that begins processing.
  4. Has an update method that updates the hash value on input.
  5. Has a getHash method that gets the final hash value.

This class has three states:

  • None : Not ready
  • Initialized : Ready to derive key data

General usage of this class is as follows:

  • Initialize PBKDF1 calculation with initialize().
  • Derive key data with getBytes().

Postcondition
  • The derived key is written to key.
Exceptions

Note
