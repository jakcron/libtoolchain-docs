Detailed Description

template<size_t T, bool byte_order_le = true, bool bit_order_le = true>
struct tc::bn::bitarray< T, byte_order_le, bit_order_le >

This struct is a literal bitarray, with configurable byte and bit endianness.

Template Parameters
Tsize in bytes of the bitarray.
byte_order_leBoolean, true: byte order is little endian, false: byte order is big endian.
bit_order_leBoolean, true: bit order is little endian, false: bit order is big endian.

This struct is meant to be used when defining written-to-disk structures, like file headers.
